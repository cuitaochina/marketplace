# Claude Code Marketplace

Claude Code 插件和扩展市场

## 概述

这个仓库包含 Claude Code 的插件、扩展和自定义工具。

## 插件分类

- 🛠️ **开发工具**: 代码生成、重构、调试工具
- 📊 **数据分析**: 数据处理、可视化、报告生成
- 🎨 **设计工具**: UI组件、样式生成、设计系统
- 🔧 **自动化**: CI/CD、部署、工作流自动化
- 📚 **文档**: 文档生成、API文档、教程
- 🤖 **AI工具**: LLM集成、智能助手、机器学习

## 安装插件

```bash
/plugin marketplace add cuitaochina/marketplace
```
